Project Name: Image Classification Model

Description:
This repository contains the necessary files and folders for training and testing an image classification model. The model aims to classify images into different categories based on their labels. Below is an overview of the folders and files included in this repository:


Folders:

Image-info-csv: Folder containing csv files providing the file names and labels of the images
logger: Folder containing logger files of the model training, data augmentation, and model architecture & hyper parameter tuning
Test: Folder containing all the test images provided
Train: Folder containing all the train images provided

sa4110_Group5_CA.ipynb: Jupyter notebook of the code for the cnn model
sa4110_Group5_CA.pdf: pdf file of the jupyter notebook and the report for the course assignment
